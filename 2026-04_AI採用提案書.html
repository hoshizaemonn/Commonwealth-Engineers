施策①　新卒向け採用コンテンツのAI量産

月2本の採用特化記事をAIで制作・公開。新卒が検索するキーワードで上位表示を狙います。

記事テーマ（例）狙うキーワードターゲット
未経験からプラント設計エンジニアになる方法 プラントエンジニア 未経験 / 新卒 理工系新卒
プラント設計エンジニアの1日【入社2年目のリアル】 プラント設計 仕事内容 / 1日 企業研究中の学生
文系からでもOK？CAD未経験で入社した先輩の話 CAD 未経験 就職 / 文系 エンジニア 文系学生
プラント設計エンジニアの年収・キャリアパス プラントエンジニア 年収 / 将来性 待遇重視の学生
CEC様にお願いしたいこと: 入社1〜3年目の社員に15分のヒアリング（月1回程度）。AIが記事化するので、話してもらうだけでOKです。

施策③　X・Instagram開設 + AI投稿テンプレート

新卒が企業研究で必ず見るSNSに、CEC様の存在を作ります。

媒体投稿内容（AI生成）頻度
X 現場風景 / 社員紹介 / 技術トピック / 採用情報 週3〜5回
Instagram 社員の1日 / オフィス風景 / プロジェクト紹介 週2〜3回
運用の仕組み: 毎月30本分の投稿文+画像キャプションをAIで一括生成してお渡しします。CEC様は写真を撮って投稿するだけ。1投稿5分で完了です。

施策④　AI採用PR動画

社員の写真・現場写真から、30〜60秒のショート動画をAIで自動制作。SNSやWantedlyで使えます。

動画テーマ内容使用先
「CEC 1分でわかる会社紹介」実績・現場・人の3要素で構成X / Instagram / Wantedly
「先輩エンジニアの声」写真+テロップ+ナレーションで構成採用LP / SNS
「プラント設計の仕事とは」現場写真で仕事の魅力を伝えるYouTube / SNS

※ 弊社ではRUNBIRD様・ミチルシリカ様でAI動画制作パイプラインを運用中。同じ仕組みをCEC様向けに展開します。
