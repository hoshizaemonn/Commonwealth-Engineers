# 記事作成ガイド

## 📁 フォルダー構成

```
articles/
├── 客先コメント/          # 客先からのコメントPDFを保管
├── 2025-12/              # 月別フォルダー（例：2025年12月の記事）
├── 2026-01/              # 月別フォルダー（例：2026年1月の記事）
├── images/               # 記事で使用する画像
└── README.md            # このファイル
```

---

## 🚀 記事作成手順

### ステップ1：事前確認

記事を作成する前に、**必ず**以下を確認してください：

1. **客先コメントの確認**
   - `articles/客先コメント/` フォルダーを確認
   - 最新のコメントPDFがあれば内容を確認
   - 禁止表現リストを確認（`knowledge/article-writing-guidelines.md`参照）

2. **ガイドラインの確認**
   - `knowledge/article-writing-guidelines.md` を読む
   - 禁止表現リストを確認

### ステップ2：記事作成

1. **適切なフォルダーに記事を作成**
   - 月別フォルダー（例：`2026-01/`）を作成
   - ファイル名：`YYYYMMDD_記事名.md`（例：`20260101_newyear.md`）

2. **チェックリストを使用**
   - `templates/article-creation-checklist.md` を使用
   - 各項目を確認しながら作成

3. **2つの形式で作成**
   - Markdown版（`.md`）：編集・確認用
   - WordPress用HTML版（`_wp.html`）：WordPressに貼り付け用

### ステップ3：最終確認

1. **チェックリストで確認**
   - `templates/article-creation-checklist.md` の全項目をチェック
   - 禁止表現が含まれていないか確認

2. **リンターエラーの確認**
   - エラーがないことを確認

### ステップ4：提出

1. **メール作成**
   - `communications/` フォルダー内のメールテンプレートを参照
   - 記事案提出メールを作成

2. **ファイル添付**
   - Markdown版（`.md`）
   - WordPress用HTML版（`_wp.html`）

---

## 🚫 重要な注意事項

### 必ず確認すべきこと

1. **客先コメントフォルダー**
   - 記事作成前に **必ず** `articles/客先コメント/` フォルダーを確認
   - 新しいコメントがあれば、内容を反映

2. **禁止表現の回避**
   - 「プロの設計者」という表現は使用禁止
   - 代替表現：「実務で使用している」「実践的な」など
   - 詳細は `knowledge/article-writing-guidelines.md` を参照

3. **表現の統一**
   - 謙虚な表現を使用
   - 専門用語を避ける
   - 一般の方にも分かりやすい表現を心がける

## 🔄 客先コメント反映プロセス

新しい客先コメントを受領した場合：

1. **コメントPDFを保存**
   - `articles/客先コメント/` フォルダーに保存
   - ファイル名：`記事名_コメント.pdf`

2. **内容を確認・反映**
   - PDFの内容を確認
   - 指摘事項をリストアップ
   - `knowledge/article-writing-guidelines.md` を更新

3. **既存記事の修正**
   - 指摘があった記事があれば修正
   - 修正内容を記録

4. **ガイドラインの更新**
   - 新しい禁止表現を追加
   - チェックリストを更新

---

## ✅ 記事作成のベストプラクティス

1. **常に客先コメントを確認**
   - 記事作成前に必ず確認
   - 新しいコメントがあれば反映

2. **チェックリストを使用**
   - 毎回チェックリストを使用
   - 見落としを防ぐ

3. **過去記事を参考に**
   - 同じスタイル・トーンを保つ
   - 一貫性を重視

4. **2つの形式で作成**
   - Markdown版：編集・確認用
   - HTML版：WordPress投稿用

5. **最終確認を徹底**
   - 禁止表現が含まれていないか
   - 誤字脱字がないか
   - SEO設定が適切か

## 📞 質問・不明点

記事作成に関する質問や不明点がある場合：

1. まず `knowledge/article-writing-guidelines.md` を確認
2. 過去記事を参考にする
3. 上記で解決しない場合は、チームに確認
