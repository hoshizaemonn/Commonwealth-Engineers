**受領日：** 2025年12月（推定）

#### 主な指摘事項

1. **「プロの設計者」という表現について**
   - NG表現：「プロの設計者が実際に使用している図面」
   - 推奨表現：「実務で使用している図面」
   - 理由：客先がこの表現を好まない

2. **その他の修正事項**
   - 「実務に直結する」→「実践的な課題」など、より適切な表現に変更

- ✅ ガイドラインに反映済み（`knowledge/article-writing-guidelines.md`）

## 📝 新しいコメント受領時の手順

1. **PDFファイルを保存**
   - ファイル名：`記事名_コメント.pdf`
   - このREADMEに記録を追加

2. **内容を確認**
   - 指摘事項をリストアップ
   - 禁止表現があれば抽出

3. **ガイドラインを更新**
   - `knowledge/article-writing-guidelines.md` を更新
   - 禁止表現リストに追加
   - チェックリストを更新

4. **既存記事を修正**
   - 指摘があった記事があれば修正
   - 修正内容を記録

## 🚫 現在の禁止表現リスト（要約）

詳細は `knowledge/article-writing-guidelines.md` を参照してください。

- 「プロの設計者」「プロの○○」関連の表現
- 「成功裏に完了」などの断定表現
- 専門用語の多用（「セッション数」など）
