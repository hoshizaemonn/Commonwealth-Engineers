# コモンウェルスエンジニアーズ様 Webサイト分析プロジェクト

## 📋 プロジェクト概要

- **クライアント**: 株式会社コモンウェルスエンジニアーズ
- **対象サイト**: https://cectokyo.com/
- **目的**: 
  - サイトのアクセス向上
  - 問い合わせ件数の増加
  - 採用問い合わせの増加

---

## 📁 フォルダ構成

```
├── data/           # GA4・Search Consoleの元データ（CSV）
├── reports/        # 月次レポート（Markdown）
├── analysis/       # 分析メモ・発見事項
├── communications/ # メール・やり取りの記録
├── instructions/   # エンジニア向け指示書
├── templates/      # テンプレート集
├── knowledge/      # ナレッジベース
└── kpi-history.md  # KPI推移まとめ
```
